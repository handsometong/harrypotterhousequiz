## Development
To run this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/handsometong/harrypotterhousequiz.git
   ```
2. Install dependencies:
   ```bash
   cd harry-potter-house-quiz && npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```
